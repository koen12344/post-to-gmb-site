  1. ABOUT THE PLUGIN

Our Plugin (product) “Post to Google My Business” has these features;

    • Create, edit or delete posts without having to visit your Google My Business page
    • Quickly publish your latest WordPress posts to GMB using the Auto-post feature
    • Network- and site-level Multisite support
    • Uses Freemius to handle payments, statistics and distribution
    • Support publishing to GMB from external apps (such as Windows Live Writer, Zapier, Integromat, ManageWP, InfiniteWP, MainWP etc)
    • Uses official Google My Business API to create posts on Google My Business locations. The users must give full business. Manage permission when authorizing for the plugin to work (there is no lower permission level unfortunately). User email and API tokens are stored securely.
    • Developer friendly. Uses the latest built-in WordPress functions and has various actions/filters to hook into.
    • Uses built-in WordPress functions for easy translation.
    • Gutenberg compatible
